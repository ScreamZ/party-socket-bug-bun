# party-socket-bug-bun

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run server.ts
bun run client.ts
```

Comment or uncomment first client import line.

This project was created using `bun init` in bun v1.0.25. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
